Module MPElectricsGE

MPElectricsGE API.

  • applyElectrics(...) -- internal access

  • MPElectricsGE.handle(...) -- external access

Local Functions

tick ()
Called on specified interval by MPUpdatesGE to simulate our own tick event to collect data.
sendElectrics (data, gameVehicleID)
This function is called by VE lua to send the data from VE to GE and then to the server. INTERNAL USE

Parameters:

  • data table The electrics data from VE
  • gameVehicleID number The vehicle ID according to the local game
applyElectrics (data, serverVehicleID)
This function serves to send the electrics data from GE to VE for application

Parameters:

  • data table The data to be applied as electrics
  • serverVehicleID string The VehicleID according to the server.
handle (rawData)
The raw message from the server. This is unpacked first and then sent to be applied according to message code.

Parameters:

  • rawData string The raw message data.
