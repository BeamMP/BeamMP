imageButton (texID, size, color, activeColor, hoveredColor)
Creates an image button with the specified texture ID, size, and colors using IMGUI.

Parameters:

  • texID number The ID of the texture to use for the button.
  • size number The size of the button.
  • color table (optional) The color of the button. Defaults to the ImGui button color.
  • activeColor table (optional) The color of the button when active. Defaults to the ImGui active button color.
  • hoveredColor table (optional) The color of the button when hovered. Defaults to the ImGui hovered button color.

Returns:

    boolean Returns true if the button was clicked, false otherwise.
