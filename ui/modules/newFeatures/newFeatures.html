BeamMP v4.0 Released!!!

Woohoo, we made it this far. Now with the acheviement of us reaching 400+ concurrent players here is BeamMP version 4.0! So what has changed? Well here goes:

  • New ModSync system is now inplace. This has been a near rewrite of the server, launcher and client mod. However you should now be able to enjoy faster downloads.
  • New features screen that will be updated with each of our updates and changes as we go!
  • Slightly revised position sync system.
  • We now no longer require you to have discord!!!
  • The mod is now much more closely aligned with Singleplayer which hopefully will provide you with a more streamlined experience.
  • UI elements will be automatically added when in a Multiplayer session just incase you do not have them already.
  • The UI system is now broken up into its own profile which means you can now have a multiplayer app layout and a singleplayer app layout!
  • BeamMP now operates out of your normal user profile directory which means your controls, UI app preferences and other settings will be automatically synced with BeamMP.
  • We had the exterminators in. Its was time consuming but most of those pesky bugs are gone now ready for the cold winter nights.
So we think that sums up this update. Hopefully you are as excited about this update as we are!
