Local Functions

tpPlayerToPos (targetPos)
Teleports the players vehicle to a specified position.

Parameters:

  • targetPos vec3 The target position to teleport the player to. Format: {x, y, z}
getPlayerNames ()
Returns the names of all players in the current session.

Returns:

    table names A table where the key is the username and the value is an owned vehicle ID (can be ignored).
getTranslations (_, content)
Retrieves translations from a JSON file and caches them?

Parameters:

  • _ any Unused parameter.
  • content string The content of the JSON file.
