tableDiff (old, new)
Compares two tables and returns the differences between them.

Parameters:

  • old table The first table to compare.
  • new table The second table to compare.

Returns:

  1. table diff A table containing the differences between the two tables.
  2. table o A table containing the values from the first table that are different from the second table.
  3. table n A table containing the values from the second table that are different from the first table.
